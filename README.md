# frontend-next

Frontend Next – Sistema de Gestión Eléctrica

Esta carpeta corresponde a la siguiente iteración del frontend del
Sistema de Gestión Eléctrica.

Su objetivo es servir como base para futuras mejoras y refactorizaciones
posteriores a la presente evaluación.

Actualmente el proyecto se encuentra en una etapa inicial, considerando:

- Proyecto Next.js creado correctamente
- Configuración base con TypeScript, ESLint y Tailwind CSS
- Servidor de desarrollo funcionando correctamente (`npm run dev`)
- Estructura `app/` lista para implementación de vistas y consumo de API

## Relación con el sistema principal

Este frontend se complementa con:

- **Backend:** Django REST Framework
- **Frontend principal:** React (SPA), donde se implementan las funcionalidades completas
  como autenticación, consumo de API y formularios (CRUD).

El desarrollo funcional completo se encuentra en el frontend React, mientras que este
frontend Next.js representa una **iteración alternativa y preparatoria**.

## Observaciones finales

Este repositorio forma parte del proceso de aprendizaje y experimentación con
tecnologías modernas de frontend, manteniendo buenas prácticas de estructura,
versionado y documentación.
